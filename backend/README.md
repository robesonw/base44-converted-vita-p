# Quickstart\n\n## Environment Variables\n| Variable | Description |\n| -------- | ----------- |\n| DATABASE_URL | PostgreSQL Connection String |\n| JWT_SECRET | Secret for JWT |\n| JWT_REFRESH_SECRET | Secret for Refresh JWT |\n| PORT | Port for backend server |\n| CORS_ORIGIN | Frontend URL for CORS |\n| AI_PROVIDER | AI Provider |\n| AI_MODEL | AI Model Name |\n| AI_API_KEY | API Key for AI |\n\n## API Reference\n\n### Auth Routes\n- **POST** `/api/auth/register` - Register a new user.\n- **POST** `/api/auth/login` - User login.\n- **GET** `/api/auth/me` - Get user info.\n- **POST** `/api/auth/refresh` - Get new access token using refresh token.\n\n### Upload Route\n- **POST** `/api/upload` - Upload a file. Returns URL.\n\n### AI Route\n- **POST** `/api/ai/invoke` - Invoke AI model, verify token.\n\n### Entity CRUD Routes\n- **FavoriteMeal**\n    - **GET** `/api/favoritemeals`\n    - **POST** `/api/favoritemeals`\n    - **GET** `/api/favoritemeals/:id`\n    - **PUT** `/api/favoritemeals/:id`\n    - **DELETE** `/api/favoritemeals/:id`\n- **Feedback**\n    - Approximation for all entity routes...\n- ...\n
